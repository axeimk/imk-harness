# hunk-review — Hunk ライブセッションの操作

Hunk はターミナルの対話的 diff ビューア。操作手順とコマンドリファレンスの本体は
hunkdiff パッケージにスキルとして同梱されており、パッケージ更新で内容が変わるため、
ここには複製せず毎回同梱版を読む。

## 手順

1. `hunk skill path` を実行してスキル本体のパスを取得する
2. 出力されたパスのファイルを読み、以降はその指示に従う

`hunk` コマンドが見つからない場合は hunkdiff が未インストール。
勝手にインストールせず、ユーザーに導入（`npm install -g hunkdiff`）を確認する。
